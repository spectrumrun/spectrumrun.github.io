🇬🇧 English

Spectrum Run does not collect, store, or share any personal information from its users. The game operates solely to provide entertainment and does not process any user data.

Data Collection

  • No device data, IP address, location, or cookies are collected.

Data Usage

Since no data is collected, no data processing occurs.

Third-Party Services

The game does not integrate any third-party services that collect user data.

This game is intended for general audiences and does not knowingly collect data from children.
